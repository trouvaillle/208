## 데이터 구조

<kbd>세상에서 제일 간단한 데이터베이스</kbd>
```sh
#!/bin/bash

db_set () {
    echo "$1,$2" >> database
}

db_get () {
    grep "^$1," database | sed -e "s/^$1,//" | tail -n 1
}
```
